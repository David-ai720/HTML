Etiquetas de Documento

Las etiquetas de un documento HTML son fragmentos de código (o "tags") que estructuran y dan formato al contenido de una página web, indicando al navegador cómo mostrar los elementos. Las etiquetas básicas son <html>, <head> y <body>; otras comunes incluyen encabezados (<h1>-<h6>), párrafos (<p>), enlaces (<a>), imágenes (<img>) y saltos de línea (<br>).
Las etiquetas básicas para un documento HTML 1.1 (y posteriores) incluyen <!DOCTYPE html>, <html>, <head>, y <body>. El documento comienza con <!DOCTYPE html> para indicar su tipo, seguido de la etiqueta <html> que delimita todo el contenido. Dentro de <html>, la etiqueta <head> contiene metadatos como el título, y la etiqueta <body> engloba todo el contenido visible de la página.

Las etiquetas de documento HTML 1.2 (la versión más común de HTML es HTML5, no hay versión 1.2) incluyen la etiqueta de declaración <!DOCTYPE html>, que define el documento como HTML5, y las etiquetas principales <html>, <head> y <body>. <html> contiene todo el documento, <head> contiene metadatos como el título de la página y <head>, y <body> alberga el contenido visible de la página web.

Las etiquetas básicas para un documento HTML 1.3 (o una versión anterior similar) incluían la estructura fundamental como <html>, <head>, y <body>, junto con etiquetas para dar formato al texto y contenido como <h1> a <h6> para títulos, <p> para párrafos, <a> para enlaces, <img> para imágenes, y <hr> para líneas horizontales. También incluía elementos de texto como <b> para negrita e <i> para cursiva, aunque con el tiempo <strong> y <em> se volvieron más comunes.


Atributos form

Los atributos form 2.1 se refieren a los atributos de la etiqueta <form> en HTML que definen el comportamiento del formulario. Los más importantes son action, que especifica la URL de destino, y method, que determina el método HTTP (\(GET\) o \(POST\)). Otros atributos clave incluyen enctype para la codificación (especialmente para subir archivos) y autocomplete para controlar el autocompletado del navegador. 

Los atributos más importantes de la etiqueta <form> en HTML son action (la URL a la que se envían los datos), method (el método HTTP a usar, como GET o POST) y enctype (la codificación de los datos, esencial para enviar archivos). Para los elementos de entrada dentro del formulario, atributos clave como name (identificador del campo) y type (tipo de input como text, email, password) son cruciales.


Atributos input

El elemento HTML <input> se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.Hay disponible una amplia variedad de tipos de datos de entrada y widgets de control, que dependen del dispositivo y el agente de usuario (user agent).El elemento <input> es uno de los más potentes y complejos en todo HTML debido a la gran cantidad de combinaciones de tipos y atributos de entrada.
